Buttons

Buttons and anchor tags are styled to look the same.

By default, they do not have a background color or outline.

<a> <a class="btn">

You can style buttons individually with the class "btn"

Or in groups "btns"

<div class="btns"> <a>button 1</a> <a>button 2</a> <a>button 3</a> </div>

The code above would output this:

button 1 button 2 button 3

By default, the class "btn" makes the button have a primary background color.
Here are a few other options:

class of "btn" class of "btn success" class of "btn danger"

navigation

Standard Nav

The standard nav without any classes is set to display: flex
It will disperse it's first children evenly using justify-content: space-between

<nav class="primary"> <a>link1</a> <a>link2</a> <a>link3</a> </nav>

Outputs

link1 link2 link3

If you want to cluster buttons, you can wrap button groups in divs with a class of "flex"

<nav class="primary"> <a>Home</a> <div class="flex"> <a>link1</a> <a>link2</a> <a>link3</a> </div> </nav>

Outputs

Home
link1 link2 link3

Collapsable Nav

To have a menu that collapses at small screen widths, wrap your links in a div with the class of "collapse" and make sure the nav contains a button with the class of "collapseToggle"

<nav class="primary"> <a>Home</a> <button class="collapseToggle">Menu</button> <div class="collapse"> <a>link1</a> <a>link2</a> <a>link3</a> </div> </nav> <!-- OR --> <div class="nav primary btns"> <a>Home</a> <button class="collapseToggle">Menu</button> <div class="collapse"> <a>link1</a> <a>link2</a> <a>link3</a> </div> </div>
