Headings are capitalized by default.

For lowercase add the class "lowercase"

For uppercase, "uppercase"

To capitalize, "capitalize"

And to remove any text-transforming, add the class "nocase"

Semantic Elements

This library is about speed, which means you can write HTML without adding any classes, and it should look good.

Classes are optional.

With this being said, you should be using semantic elements for this to work correctly.

<header>

<nav>

<main>

<section>

<article>

<aside>

<footer>

Buttons

Buttons and anchor tags are styled to look the same.

By default, they do not have a background color or outline.

<a> <a class="btn">

You can style buttons individually with the class "btn"

Or in groups "btns"

<div class="btns"> <a>button 1</a> <a>button 2</a> <a>button 3</a> </div>

The code above would output this:

button 1 button 2 button 3
